# SOBA Agent Documentation

SOBA Agent is a local-first engineering agent that remembers, verifies, and leaves receipts. The current 0.6.x release
line is a stabilization track before 0.7.0: proof receipts, memory provenance, permission receipts, skill evaluation, and
the Bun/npm global install path are the release focus.
