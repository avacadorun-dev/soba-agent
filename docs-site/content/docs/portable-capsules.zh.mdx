# Portable Capsules

完整的 Portable Capsules 指南目前提供[俄语版本](/ru/docs/portable-capsules)。

该功能包括：

- `/capsule create <objective>` — 创建 `.capsule.md` 交接文件；
- `/capsule export <checkpoint-id> <path.capsule.md>` — 导出指定检查点；
- `/capsule load <path.capsule.md>` — 将胶囊作为不受信任的上下文加载。
