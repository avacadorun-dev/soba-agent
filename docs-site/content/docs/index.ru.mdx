SOBA Agent — CLI-агент кодинга с фокусом на proactive context management, самомодифицируемую архитектуру (skills)
и гибридный визуальный слой для фронтенд-разработки.

## Быстрый старт

- [**Быстрый старт**](/ru/docs/quick-start) — установка, конфигурация, все режимы CLI/TUI, инструменты, разрешения,
  очередь, persistent-сессии, Context Capsules, proactive compaction и полный lifecycle skills за 10 минут.

## Пошаговое руководство

- [**Проект шаг за шагом: профессиональный workflow**](/ru/docs/walkthrough-building-a-project) — end-to-end сценарий
  работы с агентом: постановка задачи, реализация, тесты, review gate, capsules и handoff.
- [**Remote MCP: пошаговый гайд**](/ru/docs/remote-mcp-guide) — подключение hosted Streamable HTTP MCP server,
  env-backed auth, проверка `/mcp status` и типовые ошибки.

## Сборка

- **GitHub Releases** — tagged releases собирают standalone-бинарники для macOS/Linux и прикладывают их к релизу вместе
  с SHA256SUMS. Основной способ установки для пользователей: `npm install -g soba-agent` или `bun add -g soba-agent`.
