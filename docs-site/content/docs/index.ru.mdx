SOBA Agent — local-first инженерный агент, который помнит проект, работает в ограниченных tool loops, проверяет
результат и оставляет receipts. Текущая линия 0.6.x — стабилизационная перед 0.7.0: фокус на proof receipts, memory
provenance, permission receipts, skill eval/bench/trace и надёжной установке через npm/Bun.

## Быстрый старт

- [**Быстрый старт**](/ru/docs/quick-start) — установка, конфигурация, все режимы CLI/TUI, инструменты, разрешения,
  очередь, persistent-сессии, Context Capsules, proactive compaction и полный lifecycle skills за 10 минут.

## Пошаговое руководство

- [**Проект шаг за шагом: v0.6 workflow**](/ru/docs/walkthrough-building-a-project) — end-to-end сценарий
  работы с агентом: first-run init, постановка задачи, evidence summary, diff review, MCP reload, sessions и handoff.
- [**Remote MCP: пошаговый гайд**](/ru/docs/remote-mcp-guide) — подключение hosted Streamable HTTP MCP server,
  env-backed auth, проверка `/mcp status` и типовые ошибки.

## Сборка

- **GitHub** — исходный код, issues и releases живут в
  [avacadorun-dev/soba-agent](https://github.com/avacadorun-dev/soba-agent).
- **GitHub Releases** — tagged releases собирают standalone-бинарники для macOS/Linux и прикладывают их к релизу вместе
  с SHA256SUMS. Основной способ установки для пользователей: `npm install -g soba-agent` или `bun add -g soba-agent`.
