# Portable Capsules

The full Portable Capsules guide is currently available in [Russian](/ru/docs/portable-capsules).

This feature covers:

- `/capsule create <objective>` — create a `.capsule.md` handoff file;
- `/capsule export <checkpoint-id> <path.capsule.md>` — export a specific checkpoint;
- `/capsule load <path.capsule.md>` — load a capsule as untrusted context.

English translation coming soon.
