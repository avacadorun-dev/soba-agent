# SOBA Agent 文档

SOBA Agent 是 local-first 工程代理：记住项目、在有边界的工具循环中工作、验证结果并留下 receipts。当前 0.6.x
线是 0.7.0 之前的稳定化路线，重点是 proof receipts、memory provenance、permission receipts、skill eval/bench/trace
以及可靠的 npm/Bun 全局安装。
