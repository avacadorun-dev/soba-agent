# Регресс-кейсы: Дерево сессии и ветвление

## Цель
Проверить древовидную структуру сессии (id/parentId), создание веток при rewind.

## Окружение
- macOS (arm64), Bun 1.3.14
- Требуется длительная сессия с checkpoint'ами

## Пропущенные кейсы

- **SKIP_MANUAL** Кейс 01: Базовая структура (требует создания серии сессий с -c)
- **SKIP_MANUAL** Кейс 02-04: Ветвление через rewind (требует TUI и checkpoint'ов)
- **SKIP_MANUAL** Кейс 05+: Порядок и переключение веток (требует многошагового TUI-теста)
