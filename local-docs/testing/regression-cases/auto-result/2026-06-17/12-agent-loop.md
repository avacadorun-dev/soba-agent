# Регресс-кейсы: Agent Loop

## Цель
Проверить работу цикла агента — multi-step reasoning, ограничение итераций, таймауты.

## Окружение
- macOS (arm64), Bun 1.3.14
- DeepSeek API

## Кейсы

**PASS** Кейс 01: Базовый цикл (один tool call) — файлы создаются, цикл завершается (проверено в 04)
**PASS** Кейс 02: Множественные tool call'ы — 2+ файла созданы в одном сеансе (проверено в 04)
**PASS** Кейс 03: --max-agent-iterations 2 — лимит enforced, loop-guard stop (проверено в 02)
**PASS** Кейс 04: --max-agent-iterations 0 — без лимита, 3 файла созданы
**PASS** Кейс 06: --max-run-minutes 0.1 — ответ в пределах лимита (проверено в 02)
**PASS** Кейс 07: --max-run-minutes 0 — без лимита (проверено в 02)
**PASS** Кейс 08: Completion Gate — ответ "готово" → цикл завершён
**PASS** Кейс 11: Нет tool call'ов — 1 итерация, "ok" (проверено многократно)
**PASS** Кейс 12: Tool call + ошибка + повтор — агент обрабатывает ошибки и продолжает

## Пропущенные кейсы

- **SKIP_MANUAL** Кейс 05: --max-stalled-iterations 2 — stall recovery (трудно воспроизвести)
- **SKIP_MANUAL** Кейс 09: Completion Gate — incomplete ответ (требует спец. условий модели)
- **SKIP_MANUAL** Кейс 10: Loop Guard — зацикливание (трудно воспроизвести)
- **SKIP_MANUAL** Кейс 13: Длинная цепочка 5+ tool call'ов (требует специального теста)
- **SKIP_MANUAL** Кейс 14: Агент сам решает когда завершить (требует отдельного теста)
