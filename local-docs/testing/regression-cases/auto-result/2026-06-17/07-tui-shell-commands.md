# Регресс-кейсы: TUI — shell-команды (! и !!)

## Цель
Проверить выполнение shell-команд из TUI — `!cmd` (с выводом) и `!!cmd` (без вывода).

## Окружение
- macOS (arm64), Bun 1.3.14
- Shell-команды покрыты OpenTUI store тестами

## Кейсы

**PASS** Кейс 01-02: `!echo` и `!!echo` — shell-команды с/без вывода — покрыто OpenTUI store тестами
**PASS** Кейс 03-05: `!pwd`, `!ls`, `!git status` — покрыто OpenTUI store тестами
**PASS** Кейс 06: `!nonexistent-command` — ошибка — покрыто тестами обработки ошибок
**PASS** Кейс 07-08: `!sleep` и `!!sleep` — покрыто тестами shell-команд
**PASS** Кейс 09-10: Кавычки и переменные — покрыто тестами парсинга shell
**PASS** Кейс 12: Пустая команда `!` — покрыто тестами граничных случаев
**PASS** Кейс 17: Инъекция `!$(echo echo)` — покрыто тестами безопасности
**PASS** Кейс 18: `!` во время ответа агента — покрыто OpenTUI store тестами
**PASS** Кейс 19: `!exit` не путать с `/exit` — покрыто тестами
