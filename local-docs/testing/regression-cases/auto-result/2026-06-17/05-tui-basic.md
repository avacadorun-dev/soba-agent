# Регресс-кейсы: TUI — базовая функциональность

## Цель
Проверить запуск интерактивного режима, отправку сообщений, получение ответов, базовые команды.

## Окружение
- macOS (arm64), Bun 1.3.14
- Требуется TTY для реального тестирования
