# Регресс-кейсы: Инструменты агента (tools)

## Цель
Проверить каждый из 7 инструментов агента: ls, read, write, edit, bash, checkpoint, activate_skill.

## Окружение
- `.soba` собран
- Временная директория для тестов

---

## Кейс 01: `ls` — листинг директории

**Шаги:**
1. `.soba "Выполни ls в текущей директории и назови первый файл по алфавиту"`

**Ожидаемый результат:** Агент вызывает `ls`, получает список файлов, называет первый.

**Критерий PASS:** Tool call `ls` выполнен, результат корректен.
