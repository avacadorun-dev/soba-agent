# Manual test: остановка процесса и standalone-бинарник

## Остановка процесса без остановки агента

1. Запустить интерактивный SOBA: `bun run dev -- -i`.
2. Попросить агента запустить долгоживущую команду, например локальный dev-сервер.
3. Когда в статусе показано выполнение `bash`, нажать `Ctrl+C`.
4. Убедиться, что процесс завершён, в ленте появилось сообщение об остановке процесса, а агент продолжил текущий ход.
5. Нажать `Ctrl+C`, когда tool не выполняется. Убедиться, что текущий ход отменён или приложение завершилось в idle.

## Бинарник macOS Apple Silicon

1. Выполнить `bun run build:binary:mac-arm64`.
2. Проверить `file dist/bin/soba-darwin-arm64`: архитектура должна быть `arm64`.
3. Из директории вне репозитория выполнить:
   `/absolute/path/to/dist/bin/soba-darwin-arm64 --help`.
4. Убедиться, что help локализован и рядом с бинарником не требуются `node_modules`, `locales` или `bunfig.toml`.
5. Передать бинарник на другой Apple Silicon Mac, выполнить `chmod +x soba` и проверить `./soba --version`.
