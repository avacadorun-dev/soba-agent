# Manual Test Run: Direct Shell Shortcuts

| ID | Действие | Ожидаемый результат | Результат |
|---|---|---|---|
| DS-1 | Ввести `!printf 'visible\n'` | В ленте виден bash tool block и `visible` | |
| DS-2 | Ввести `!!printf 'hidden\n'` | Видны запуск и завершение bash, `hidden` отсутствует | |
| DS-3 | Во время работы агента ввести `!pwd`, затем `!!bun test` | Обе команды видны в очереди с правильными префиксами и выполняются FIFO | |
| DS-4 | Ввести `!sleep 60`, затем нажать `Ctrl+C` | Процесс остановлен, TUI остаётся рабочим | |
| DS-5 | Ввести `!git status --short` | Команда выполняется без approval prompt и без запроса к модели | |
