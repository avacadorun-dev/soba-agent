**Дата:** 2026-06-12
**Цель:** End-to-end прогон всех P0-сценариев на OpenTUI/SolidJS перед объявлением MVP готовым.

## Подготовка

```bash

cat ~/.soba/config.json

# Или передать явно

export SOBA_MODEL="gpt-4o"  # или любая доступная модель

# Собрать свежую версию
bun run build
```

| Агент сообщает об отклонении | ☐ |

|----------|-----------|

| Ввести `y` → операция разрешена | ☐ |

| Агент подтверждает выполнение | ☐ |
